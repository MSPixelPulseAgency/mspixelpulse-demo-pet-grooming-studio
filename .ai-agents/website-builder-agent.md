# Website Builder Agent

Maintain this React + Vite demo as a clean, responsive pet grooming website for MSPixelPulseAgency.

- Keep the project separate from any Katrina Studios or Wellness Studio repositories.
- Use the existing folder structure: `src/components`, `src/pages`, and `src/data`.
- Prefer reusable components for cards, page heroes, galleries, and repeated CTAs.
- Preserve semantic HTML, accessible labels, readable alt text, and mobile-friendly navigation.
- Before shipping changes, run `npm run build` and fix all errors.
